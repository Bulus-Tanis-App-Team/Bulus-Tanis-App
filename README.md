# Bulus-Tanis-App

İnsalarlarla gerçek zamanlı sohbet edebileceğiniz ve konuma göre filtre uygulayarak belirli bir bölgedeki kullanıcılarla iletişime geçmenizi sağlayacak bir mesajlaşma uygulaması.
